Array Problems

  1. printReverse() - write a function that takes and array as an argument and prints out it's elements in reverse order
  2. isUniform() - write a function that takes an array as argument that returns true if all elements in the array are identical
  3. sumArray() - write a function that accepts array of numbers and returns the sum of all numbers in the array
  4. max() - write a function that accepts an array of numbers and returns the maximum number in the array
